Mönac – Skill Acquisition Platform
Mönac is a modern, user-centric web interface designed to streamline the journey from curiosity to mastery. By categorizing the educational landscape into structured pathways, it helps learners find the right niche—whether it's formal university training, vocational skills, or specialized education.

🚀 Live Demo
[[Live Link](https://monac-agency-88ojrfquw-asmitas-projects-9ea0957c.vercel.app/)]

✨ Features
Structured Learning Pillars: Organizes education into 6 clear categories: Formal, Informal, Non-formal, Vocational, Distance, and Special Education.

Intuitive User Journey: A simplified three-step onboarding process (Create Account → Purchase Lessons → Start Learning) to minimize friction.

Responsive UI: A high-contrast, warm color palette paired with a mobile-first grid layout for seamless learning on any device.

Modern Aesthetics: Custom illustrations and clean typography designed to improve user retention and engagement.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (using Flexbox & CSS Grid for layout)

Interactivity: JavaScript

Design Philosophy: User-Centric Design (UCD) with a focus on accessibility and clear Call-to-Actions (CTAs).
